

Shift-s will bring up menu and can reset the origin. 
Shift-A - Shortcut for add an object. 
T - Toggles tools panel for movement, etc. 
G - Grab
R - Rotate - Select and number of degrees
SHIFT - D - Duplicate
S - Scale
Square on tools means all but that one. 
CTRL - Select can deselect
Shift - X, Y, Z is like the square.
G - X, Y, Z will constrain movement to that axis. 
SHIFT - Select will add to your selection but the most recent is active. 
ALT - Left Click is supposed to allow you to change them all but that did not work for me. 
ALT-E is like extrude but with options. 
CTRL-SHIFT will allow you to add all between two items. 
M - To Move to new collection.
K - Knife
	Bisect does not have a keyboard shortcut but looks to be better for some things. 
SHIFT-D - Duplicate
ALT-D - Linked Duplicate
. - Recenters view. 

After Selecting a bunch do a shift select on the one you want to make active. 
	

X,Y,Z to go in that direction
SHIFT-X,Y,Z to not go in that direction. 
There is a little arrow that you can use to set position of item.  or ( N )

Del - Dissolve edge to get rid of lines in faces. 


GG can select a loop cut and move it. 
I - Inset - II Inset individual


0 - Camera View
1 - Numpad = Front view
3 - Side View
5 - Orthographic mode
7 - Top View

1 2 3 - Vertex, Edge, Face but not emulate numpad at the same time. 
E - Extrude
TAB - To go back to object mode.

Select an item and F2 will allow you to rename it. 

Right click an item in collection and hit M to create a new collection.

Item selected on right.  CTRL-A to apply scale. 

CTRL-B to bevel - V to do vertices.
